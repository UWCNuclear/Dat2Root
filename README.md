# Dat2Root
Converts an ASCII file to a ROOT histogram.

Run in the command line with:

grsisort -l Dat2Root.C


or with:


grsisort -l

.x Dat2Root.C


or with:


grsisort -l

.L Dat2Root3.C

Dat2Root3("somefilename.dat")


Also, all the above 'grsisort' can be replaced with 'root' - and it should work just fine!
